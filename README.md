## Run Locally

Clone the project

```bash
  git clone https://github.com/anthonypz/drum-machine.git
```

Go to the project directory

```bash
  cd drum-machine
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
